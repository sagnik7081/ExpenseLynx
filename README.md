## 🚀 Quick Start

```bash
# Clone & Install
git clone https://github.com/sagnik7081/ExpenseLynx
cd ExpenseLynx
pip install -r requirements.txt

# Add your API key
echo "GROQ_API_KEY=your_key_here" > .env

# Launch
python app.py
```

Visit `http://localhost:5000` 🎉
